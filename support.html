üêõ Common Issues

Installation Errors

If Wabbajack throws errors during installation:

  • Click "Retry" to resume - this is normal for large texture packs
  • Ensure you have enough disk space (140+ GB free)
  • Check that Fallout 4 is updated to the latest version
  • Verify you have all required DLCs installed

Performance Issues

For low FPS or stuttering:

  • Ensure the modlist is installed on an SSD (not HDD)
  • Set pagefile to 20-30GB
  • Lower in-game settings to High or Medium presets
  • Apply the MCM preset as described in the README
  • Disable ENB/ReShade if on lower-end hardware

Crashes

If the game crashes:

  • Do not open MCM menu before completing character creation
  • Ensure you have 32GB RAM installed
  • Check that pagefile is properly configured
  • Verify game files through Steam
