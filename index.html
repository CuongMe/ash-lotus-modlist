üì¶ Installation

Ash Lotus can be installed easily through Wabbajack, the automated modlist installer.

  1. Download the latest version of Wabbajack from wabbajack.org
  2. Download the .wabbajack file from Nexus Mods
  3. In Wabbajack, select Install from Disk and choose the downloaded file
  4. Follow the on-screen instructions to complete installation

Note: If Wabbajack throws errors after installation, click retry to resume downloading. This is normal for large texture packs.
